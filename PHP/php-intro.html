How php works

When you go to a webpage on the internet you're essentially saying to the global network, "Hey, can you pass me this file, at this EXACT location?". Otherwise known as a URL.
The server hosting that URL, then looks at the file you've requested, deciphers the code, then displays the webpage's content dependent on the answers to the code.
php is a special bit of code that tells the server to retrieve content from the SQL database and replace the php code with content from the database dependent on the answer to the php code.
