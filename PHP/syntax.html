PHP-syntax

Opening up PHP

PHP-syntax must be opened with a <?php and closed with a ?>

Long-hand, but more accepted

<?php
     // code in here
?>

Short-hand

<?
     // code in here
?>

echo & strings

Now that we know how to open our php tags, it's time to write our first bit of php code.

In PHP, when we want to write anything in the server we can use the echo construct. This is not a function, even though it can be written as one.

Now because this is a string we had to wrap it in "quotation marks" this tell the server this is just text, don't do anything fancy with it.

<?php
     echo "Hello World";
?>

One of the great features about PHP is you can put it anywhere in your HTML, and that'll be where it's placed.

Semicolons

When you I use them?

  • After every single line
  • After every single statement

JavaScript and jQuery are quite forgiving when it comes to Semicolons, but PHP is not. If you forget a single semicolon your code will not work.
