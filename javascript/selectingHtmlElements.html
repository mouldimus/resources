selectingHtmlElements

Using the document.getElementById, we can select the HTML element with the ID tag "test", using JavaScript.

Essentially we're saying to the JavaScript, hey in this document there's an element with the Id "test" can you get it for us.

Once we then have the element, we can assign it as a variable and start using JavaScript to modify it.

And trust me there's alot more ways of modifying stuff in JavaScript.



Side note: if you're just changing the inner text of the HTML element, use .innerText instead of .innerHTML as the browser doesn't have to waste time rendering HTML.





var test = document.getElementById("test");


//This is a way of modifying the HTML inside an element
test.innerHTML = "<u>selecting</u>HtmlElements";
