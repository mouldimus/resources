text-align-last

So the text-align-last attribute aligns the last line of text.

Example

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Asperiores dolore voluptatibus harum soluta error ex incidunt beatae ipsa magnam doloremque nisi explicabo consequuntur amet, dolor mollitia possimus illo sequi magni fugiat quis accusantium, sapiente atque quia. Quae velit dolorem ex explicabo tempore dolor maxime repellat!

p{
text-align: justify;
text-align-last: justify;
}
