transform-rotate

Using rotateX(), I can rotate elements as if they were 3D

.box{
width: 150px;
height: 150px;
background: #000;
margin: 10px auto;
transition: 1s all cubic-bezier(0.77, 0, 0.175, 1);
}
.box:hover{
background: #0000ff;
height: 175px;
width: 175px;
}

rotateX(180deg)

.box1:hover{transform: rotateX(180deg);}

rotateY(180deg)

.box2:hover{transform:rotateY(180deg);}

rotateZ(90deg)

.box3:hover{transform: rotateZ(90deg);}

Side note: you can use the "all" attribute in transitions to transition all the attributes that can change. Rather than just the one.
