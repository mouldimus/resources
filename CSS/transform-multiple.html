transform-multiple

To add multiple tansformations to an element or element state just add the next transformations in with it.

Make sure not to add commas inbetween the transformation attributes. They are not are applied in no particular order, not in a list.

side note: comma seperators in CSS stand mean they are in an order list, and with out means they are in an unordered list.

Example

transform: scale(2) translate(20px, 20px) skew(5deg);
