custom-fonts

Installing custom fonts into a website is fairly easy, just tedious.

You start off by creating a @font-face rule for each different font-style and font-weight.

Then inside each on of those rules, you'll need to name the font-family and specify its font-weight and font-style.

Finally, insert the url into the src of the font-style and font-weight required.

Example

@font-face {
font-family: "roboto";
font-weight: 900;
src: url(fonts/Roboto/Roboto-Black.ttf);
}
@font-face {
font-family: "roboto";
font-weight: 900;
font-style: italic;
src: url(fonts/Roboto/Roboto-BlackItalic.ttf);
}
