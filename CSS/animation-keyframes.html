animation-keyframes

An animation requires you to state what's being done at what point during the animation.

To do that we use the keyframes rule. Which in an animation would look something like:

@keyframe rule

@keyframe animation-name{
0%{
font-size: 10px;
}
50%{
font-size: 200px;
}
100%{
font-size: 10px;
}
