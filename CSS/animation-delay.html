animation-delay

This one again was pretty easy. It's the same as the transition attribute with extra steps.

Example

.box{
width: 150px;
height: 150px;
background: #000;
margin: 0 auto;
animation: blinking 1s ease 2s 2 forwards;

}
@keyframes blinking {
0%{opacity: 1;}
50%{opacity: 0;}
100%{opacity: 1;}
}
