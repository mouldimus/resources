box-sizing

So box-sizing is a way to change how the browser calculates the size of the box.

    Either from the
  • inside
  • outside
So box-sizing: content-box; measures the box width from the inside of the box, normally called the content box.
Whereas box-sizing: border-box; measures the box width and height from the outside, otherwise called the border box.

For Example

.box {
border: 1px solid #000;
margin: 50px 0;
width: 300px;
padding: 20px;
background-color: #999;
}
.box1 {
box-sizing: content-box;
background-origin: content-box;
}
.box2 {
box-sizing: border-box;
background-origin: border-box;
}
