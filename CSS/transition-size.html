Transitions are pretty simple really.

Start off by creating two things, in different states, you want to transition together.

Example

.box{
height: 50px;
width: 50px;
border: 1px solid #000;
background: #0000ff;
display: inline-block;
}

.box:hover{
height: 250px;
width: 250px;
}

Then add the transition attribute to the default state of the element. Using the shorthand, transition, you can put more than one attribute together in one line of code.

The main ones you need to worry about are:

  1. Length of time you want the transition to last
  2. The attribute you are transitioning
  3. The speed curve at which it flows through the transition

Example

.box{
transition: 325ms height linear,
325ms width linear;
/* time attribute speed-curve */
}
