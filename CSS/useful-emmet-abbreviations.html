Bonus Lesson

So I've been looking at how some other editors write code online and they're a hell of a lot faster than me (to be expected) and my tutor.

So today I'll be doing my own research on the subject.

Here are mine findings:

Source
  1. You can write elements without needing the < > brackets. (duh)
  2. You can tab between selected parentheses, but only if the position is highlighted by emmet.
  3. Use ! and enter/tab to fill out all the basic fundementals of a html document.
  4. The period key is shorthand for class. For example, p.this-is-class would print <p class="this-is-a-class"></p> with your cursor between the p tags. Additionally if you do not attach the period key to an element it will automatically assume its a div tag.
  5. # is shorthand for id. Works the same as the period key
  6. [] is shorthand for attribute.
  7. {} is shorthand for inside the html tags
  8. > is shorthand for a child element
  9. * is shorthand for multiply the shorthand by the number following it.
  10. + can be used to create a sibling elements.
  11. $ is used to print the next sequential number. For example, li.item$*5 would create 5 list elements, with the classes item1, item2, item3, item4 and item5 respectively.
  12. lorem prints out 229 words of Lorem Ipsum, an ancient latin piece of text used to fill in text areas on websites. You can aslo specifcy exactly how many words of Lorem Ipsum you want printed by doing lorem50
