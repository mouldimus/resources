flexbox-align-self

align-self overides other flex attributes assigned to it by default.

Here are some of the values that can be set:

flex-end
center
flex-start
stretch

(only if h/w is == auto)

align-self: flex-end;

.child{
align-self: flex-end;
}

align-self: center;

.child{
align-self: center;
}

align-self: flex-start;

.child{
align-self: flex-start;
}

align-self: stretch;

.child{
align-self: stretch;
}
