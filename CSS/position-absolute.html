position-absolute

For position:absolute; to work the parent element must be set to position:relative;
Once set the absolute element will be able to set a position inside the parent element using the following positioning codes:

  • top: 10px;
  • right: 50px;
  • bottom: 100px;
  • left: 200px;

Additionally, the calc() command can be used, in conjunction to calculate the correct positioning of the child element.

For example, I used:
top: calc(50% - 25px)
Minus 25px to find the centre of the element.
