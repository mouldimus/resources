There is a way to reorder elements using CSS.

However, all elements must be child to the display: flex; parent.

Pretty sure if one has an order value the element gets put after the other elements within the family.

For Example

.container {
display: flex;
}
.container div {
flex: 0 1 20%;
}

.box1 {
order: 4;
}
.box2 {
order: 1;
}
.box3 {
order: 2;
}
.box4 {
order: 3;
}
element 1 order: 3;
element 2 no order assigned
element 3 order: 2;
element 4 no order assigned
