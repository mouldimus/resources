background-img-sizing

There are four different ways you can change a background size. They are:

  • Cover

    • Covers the specified area with the img until either width or height hits that limit then stops there.

  • Contain

    • Contain fits all of the img in the specified area, without changing the resolution size, then repeats the img until the specified area is full.

  • px, cm, mm, inches, etc

    • Set the height and width using measurements

  • Percentage

    • Set the height and width using a percentage of the specified area.
