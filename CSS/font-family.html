font-family

Serif

Serif fonts are often used for longer bits of text like articles, because the little ticks help you keep focused.
These fonts contain serifs — small decorative strokes that protrude from the main body of the letter. Serif fonts are easier to read in physical, printed formats, as the serifs lead the viewer’s eyes from character to character. Times New Roman is a serif font.

Sans-serif

Sans-serif fonts are often used for everything else, because it looks better.

Monospace

This term refers to fonts that have equal spacing between characters. Courier is a monospace font.

Cursive

These fonts resemble handwriting. Brush Script MT is a cursive font.

Fantasy

Fantasy fonts are highly stylized decorative fonts. Luminari is a fantasy font.

MS

The MS in font names stands for Microsoft. It shows that Microsoft created this font for its digital devices. Trebuchet MS is an example.
