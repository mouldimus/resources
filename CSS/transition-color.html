transition-color

Change color on hover

Again, just like with transition-size you'll need to create an element with two different attribute values, on different element states.

Example

a{
text-decoration: none;
color: #000;
}
a:hover{
color: #0000ff;
}

Then, just like with the transition-size you'll want the define the transition-duration, transition-property and transition-timing-function.

Example

a{
transition-duration: 0.5s;
transition-property: color;
transition-timing-function: ease-in-out;
}
