animation-mini-project

In this project all we had to do was make a box move around its container, corner to corner.

Here's how I did it:

Example

.box {
width: 50px;
height: 50px;
background: orange;
position: absolute;
animation: swim 4s ease 0s 3 alternate forwards;
}
.container {
width: 100%;
height: 1000px;
background: blue;
position: relative;
}
@keyframes swim {
0% {
left: 0;
top: 0;
}
25% {
left: calc(100% - 50px);
top: 0;
}
50% {
left: calc(100% - 50px);
top: calc(100% - 50px);
}
75% {
left: 0;
top: calc(100% - 50px);
}
100% {
left: 0;
top: 0;
}
}
