transform-3D

Almost all transform values have 3D capabilites, to spot which ones just look out for the X, Y and Z axises in the values.

The ones that have 3 dimensions are:

rotate()

.container:hover .box1 {
transform: rotateX(180deg) rotateY(180deg) rotateZ(180deg);
background: blue;
}

scale()

.container:hover .box2 {
transform:scaleX(2) scaleY(2) scaleZ(2);
background: red;
}

translate()

.container:hover .box3 {
transform: translateX(20px) translateY(20px) translateZ(20px);
background: green;
}
