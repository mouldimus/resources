For Example

.parent {
display: flex;
}

.parent div {
flex-basis: 33%;
flex-shrink: 1;
flex-grow: 1;
border: 1px solid red;
}

flex-basis

flex-basis: 33%;

This specifies what you would like the elements width to be.

flex-shrink

flex-shrink: 1;

This can be filled out by a 1/0, Yes/No or True/False.

It defines whether or not the element is allowed to shrink.

flex-grow

flex-grow: 1;

This can be filled out by a 1/0, Yes/No or True/False.

It defines whether or not the element is allowed to grow.

flex

flex: 1 0 25%;

flex is the shorthand for all three.

Syntax: none | [ 'flex-grow' 'flex-shrink' || 'flex-basis' ]
