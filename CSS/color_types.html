Color Types

CSS often changes colors of many diferent things.
Here's how to do that

  • Named colors - red/blue/black....
  • Hexacolors - #000000(black)...
  • RedGreenBlue(RGB) - solid colors -
  • RedGreenBlueAlpha(RGB) - RGB, but with opacity functionality

Hexacolors

  • Colors in hex range from 0 to f
  • #000000 - black
  • #ffffff - white

RGB Colors

  • Colors in RGB range from 0 to 255
  • RGB(0,0,0) - black
  • RGB(255,255,255) - white
